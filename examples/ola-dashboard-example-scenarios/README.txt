These documents describe the 8 basic Use Cases and Scenarios of the EDSSI Activity 2.4 and 20 corresponding XML examples.
Inside the /src folder one can find the editable MS Word (.docx) and Visio (.vsdx and .vsd) files. Each team that wishes to edit this template to write their one scenarios can choose the format of their preference.
The two PDF files share the same content, one of them being in a printable form (the one extracted from the Word file).
